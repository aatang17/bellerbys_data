# Bellerbys

This repo contains two apps:

| App | Folder | Description |
|-----|--------|-------------|
| **Bellerbys Offer Database** | [`Bellerbys Offer Database/`](Bellerbys%20Offer%20Database/) | Upload PDF/image offer letters, extract data with Gemini, store in SQLite. See [Bellerbys Offer Database/README.md](Bellerbys%20Offer%20Database/README.md) for setup and run. |
| **Offer Letter Generator** | [`Offer_Letter_Generator/`](Offer_Letter_Generator/) | Generate offer letters from a DOCX template. |

**Docs:** [Railway deployment](docs/RAILWAY.md), [project notes](docs/PROJECT_NOTES.md).
